# Geometric Control — DUNR

This document describes the geometric survey control carried out at Dunrobin House, including site details, methodology, equipment and an index of the supporting documents contained in this folder.

## Method:

Survey method used:

- **TST (Total Station / Theodolite)** 

**Description:** Traditional total Station survey, establishing backsight for local grid.

**Existing Network?:** No.

## Coordinate Reference System (CRS):

Description of the coordinate reference system:

- **Local:**

**Description:** No need for results to be tied into national grid. Local CRS designated for photogrammetric network and revisit if necessary.

## Network Hierarchy:

Describe the role of this control network within the wider survey hierarchy (delete as necessary):

- **P** – Primary

**Description:** Primary contained network.

## Equipment:

Specify the equipment (e.g., make, model, serial number) used to undertake the control surveys (delete as necessary):

**Measurement:**
- **TST:** Leica TS10 Total Station

**Survey Markers / Targets:**
- **Make / Model:** Custom thermal-specific survey markers (V002) created by Neil Sutherland
- **Description:** 60x60mm, aluminium and rubber crosshair targets. FormLabs 3D-printed baseplate. Chosen for visibility in all imaging modalities.
- **Fixing / Adhesive:** Adhered with Rothbucher RMSK FIX. 

## Permanence:

Description the state of the control survey (delete as necessary):

- **Permanent**

**Description:** First test of survey marker durability. As the property has no heritage designation and permission not needed, markers have remained on the facade.
